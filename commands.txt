TRAIDER BOT — командная шпаргалка (v72)

Портфель и рынок
/now — карточки по всем парам (core сверху, stable в конце)
/portfolio — сводка по активам и % распределению
/json [<filename>] — список/скачать JSON из /data

Бюджеты (LONG core)
/budget — общая сводка по всем core монетам (блоки + Totals, Week)
/budget <SYMBOL> — сводка по одной паре
/budget schedule — интервалы текущей недели и месяца + TZ
/budget <SYMBOL>=<N> — установить weekly бюджет (целое)
/budget <SYMBOL>+<N> — прибавить к weekly бюджету (целое)
/budget spent <SYMBOL> <LEG> <N> — отметить ручное списание (LEG: OCO|L0|L1|L2|L3)
/budget reset — обнулить потраченное и ролловеры как в начало периода
/tz<+H|-H> — установить часовой пояс в часах (пример: /tz+9)

Планировщик
/sched status — статус и текущий интервал
/sched set interval=<sec> jitter=<sec> — изменить интервал и джиттер на лету
/sched tail <N> — показать последние N строк лога планировщика

Прочее
/commands — прислать этот файл
