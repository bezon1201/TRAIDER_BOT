# 73 — Fix: 'Текущая' price in confirm

- Добавлен надёжный парсер текущей цены `extract_current_usdc(...)` с каскадом: ticker.last → lastPrice → price/markPrice → mid из стакана.
- В подтверждении больше не подставляется `0.00` при отсутствии поля — показываем «Текущая: —» без Δ.
- Стрелка Δ считается от L1: `→` при |Δ| < tick, иначе `↑/↓`.
- Округление по тик-сайзу (через Decimal).

Если в тексте подтверждения использовалась строка 'Текущая: ...', она заменена плейсхолдером и теперь строится через `_apply_current_line(...)`.
