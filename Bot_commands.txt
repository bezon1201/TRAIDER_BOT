Базовые команды
/start - запустить бота и получить приветствие
/help  - показать список всех команд (этот файл)
/menu  - показать главное меню (MAIN MENU) с кнопками верхнего уровня
/coins - показать или изменить список монет (coins.json)
/metrics - обновить метрики для всех монет из списка
/metrics <SYMBOL> - обновить метрики только для указанного тикера
/rollover - пересчитать state.json для всех монет из списка
/dca start <SYMBOL> - построить DCA-сетку для указанного тикера
