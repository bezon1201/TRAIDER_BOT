Базовые команды
/start - запустить бота и получить приветствие
/help  - показать список всех команд (этот файл)
/menu  - показать главное меню (MAIN MENU) с кнопками верхнего уровня
/coins - показать или изменить список монет (coins.json)
/metrics - обновить метрики для всех монет из списка
/metrics <SYMBOL> - обновить метрики только для указанного тикера
/rollover - пересчитать state.json для всех монет из списка
/dca start <SYMBOL> - построить DCA-сетку для указанного тикера

1.43 CARD1: новая карточка MAIN MENU (Start/Stop из _grid, Market/TF/MA30 из state, Anchor/Budget из dca_config, Current ask из <SYMBOL>.json).
1.44 MAIN MENU: автообновление карточки после ввода BUDGET/LEVELS/ANCHOR и изменения списка монет (COINS).
1.45 METRICS: MAIN MENU перерисовывается после глобальной METRICS (PAIRS) и команды /metrics.

1.46 COINS: кнопка COINS показывает текущий список монет в alert и тихо обновляет MAIN MENU после успешного ввода.
1.47 MENUS: MAIN MENU перерисовывается с сохранением текущего подменю (current_menu), клавиатура не прыгает в MAIN.
1.48 BUDGET: запрет изменения конфига при активной кампании и тихое сохранение бюджета без псевдо-alert.
1.49 LEVELS: блокировка изменения уровней при активной кампании, тихое сохранение без псевдо-алертов.
1.50 ANCHOR: блокировка изменения ANCHOR при активной кампании, тихое сохранение без псевдо-алертов.
1.51 ON/OFF: блокировка переключения DCA при активной кампании (alert, без открытия диалога подтверждения).
1.52 CONFIG: блокировка входа в подменю CONFIG при активной DCA-кампании.
1.53 DCA_CONFIG: расширен формат dca_config.json (anchor_mode/offset) с обратной совместимостью.
